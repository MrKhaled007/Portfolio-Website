03

A.I Chess Bot

archived

Machine learning chess engine trained on the Lichess game database to predict grandmaster-style moves from board positions. Built as a university exam project to explore multi-class classification at scale and the practical limits of classical ML on a structured-but-enormous decision space.

  • python
  • scikit-learn
  • sgd
  • python-chess
  • lichess-bot
read more →

PGN games are parsed into (position, move) pairs, encoded as feature vectors, and fed into an SGDClassifier trained incrementally on batches too large to fit in memory. A LabelEncoder maps thousands of legal UCI moves to class targets, and the trained model plugs into the lichess-bot framework as a homemade engine. Runs locally against Lichess opponents; not currently deployed.

github ↗
04

MCP Data Cleaner

archived

First hands-on MCP server build — exposes a pandas-based data-cleaning pipeline as a tool that any MCP-compatible LLM (Claude, GPT, local models) can call to clean tabular data on demand. Built to learn the Model Context Protocol end-to-end, from server definition to tool invocation.

  • python
  • mcp
  • pandas
  • claude
read more →

The server wraps a cleaning routine that drops duplicates, imputes missing values with median for numeric columns and mode for categorical, and returns a log of every transformation applied. Deliberately kept the cleaning logic simple to focus the project on the MCP layer: tool registration with @mcp.tool(), type-annotated inputs, and structured string outputs that LLMs can parse and reason about. A CLI agent wrapper provides a non-MCP fallback for local testing.
